# Changelog

Alle wichtigen Änderungen an diesem Projekt werden hier dokumentiert.

## [1.0.0] - 2026-01-29

### Hinzugefügt
- Initiale Version des Stream Display Servers
- MQTT-Steuerung für Stream-Umschaltung
- Web-Interface für Konfiguration und manuelle Steuerung
- UniFi Protect API Integration
- Nahtlose Stream-Umschaltung ohne Schwarzbild
- Fallback-Bild bei Stream-Unterbrechung
- Automatischer Reconnect bei Verbindungsabbruch
- Bitfocus Companion Template für Streamdeck
- Ein-Befehl-Installation
- Automatischer Start als Systemd-Service
- Hardware-beschleunigte Video-Wiedergabe mit mpv
